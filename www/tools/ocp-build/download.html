ocp-build

Installation

ocp-build is distributed under GPLv3.

Install with OPAM

In OPAM, ocp-build has its own package, that actually triggers the installation of TypeRex, in which it is included: 

% opam info ocp-build
             package: ocp-build
  available-versions: 0.1, 1.99.3-beta
         description: Project manager for OCaml
You should install the latest version (here 2.00.0-beta): 
% opam install ocp-build.1.99.3-beta
The following actions will be performed:
 - install typerex.1.99.3-beta [required by ocp-build]
 - install ocp-build.1.99.3-beta
2 to install | 0 to reinstall | 0 to upgrade | 0 to downgrade | 0 to remove
Do you want to continue ? [Y/n]

=-=-= typerex.1.99.3-beta =-=-=
The archive for typerex.1.99.3-beta is in the local cache.
Extracting /home/lefessan/.opam/archives/typerex.1.99.3-beta+opam.tar.gz.
Building typerex.1.99.3-beta:
  ./configure --prefix /home/lefessan/.opam/4.00.1
  make
  make install
Installing typerex.1.99.3-beta.

=-=-= ocp-build.1.99.3-beta =-=-=
Installing ocp-build.1.99.3-beta.
and you have now ocp-build on your system: 
%  ocp-build -about
ocp-build : OCaml Project Builder
	version: 1.99.3-beta
	description:
	author: Fabrice LE FESSANT 
	license: GPLv3

Install from Sources

The sources of ocp-build are available in the TypeRex repository on GitHub. Latest versions can be found in the branch typerex2. 

git clone git://github.com/OCamlPro/typerex.git
cd typerex
git checkout typerex2
./configure
make
make install
