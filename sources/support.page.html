TypeRex is an Open-Source software

TypeRex is open-source software developed by OCamlPro and INRIA.

  • You can ask your questions about TypeRex on Stack Overflow. Make sure to tag your questions with at least TypeRex and OCaml.
  • Please report any bugs, unexpected behavior, or unclear documentation, through our Github issue tracker. Please let us know also about your most wanted features so that we can optimize our agenda
  • Fork our project on Github to contribute. We gladly welcome patches, especially the ones which improve TypeRex support on new platforms.

Technical Support

OCamlPro provides technical support on TypeRex. OCamlPro can help you if you are looking for:

  • training on TypeRex
  • targeted development efforts (typically for supporting new platforms)
  • long-term commercial support
